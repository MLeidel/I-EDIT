I-EDIT Quick Help

SPECIAL KEY SHORTCUTS: Ctrl-s => Save Ctrl-r => Render in frame or new tab Ctrl-Enter => Insert "<br>" at cursor position Ctrl-Space => Insert "&nbsp;" at cursor position Alt-t => insert mm/dd/yyyy date at cursor position Alt-z => Insert extended text keyed on SELECTED text (tags.js)   Default: make tag <SELECTED></SELECTED> text Alt-x => view all commands in the tags.js file Alt-a => Insert zen wrap command for SELECTED text   prompt = tag => formats selected text to text   prompt = scase => converts selected text to SentenceCase   prompt = $ => formats selected text to ${text} template string   prompt = * | ** | _ | __ => wraps selected text Alt-w => Repeat last Alt-a with new selected text Alt-c => launch Clips pick window (uses webdev.dat) Alt-h => this help page Esc => toggle tools menu USEFUL "ACE" DEFAULT SHORTCUTS: Ctrl-Shift-Left => select word to left Ctrl-Shift-p => select to matching bracket Ctrl-p => go to matching bracket Ctrl-f => find, find all Ctrl-h => replace, replace all Ctrl- , Ctrl+ => font smaller , larger Ctrl-/ => toggle (line or group) comment SOME HOW-TOs: Save As - Rename the file in the input box and Save. Create Directory with '/' - Enter the new directory name as /newdirectory into the file input box and click Save button. Delete file or directory - Del button deletes current file. However, if the file input box is blank, then directory is deleted. Delete with a file mask - Enter the file mask into the file input box and click Del button. Copy file from another directory with '+' - Enter fullpath of the file to copy as +fullpath into the file input box and click Save button. Copy current file to another directory - Switch to new directory and click Save button. Clone directory to a new directory with '>' - Enter the new directory name as >newdirectory into the file input box and click Save button. existing directory and contents will be copied to newdirectory Minify Javascript file with '<' - Place an < before the filename: <file.js in the file input box and click Save button.
